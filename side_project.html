俄羅斯方塊

HTML5 俄羅斯方塊遊戲

HTML5 JavaScript Canvas CSS3

這是我用純JavaScript和HTML5 Canvas技術開發的俄羅斯方塊遊戲。實現了經典的遊戲玩法，包括方塊旋轉、快速下落、消行計分等功能。遊戲界面採用了響應式設計，可以在桌面和移動設備上流暢運行。特別加入了自定義主題和音效，提升遊戲體驗。

開發這個項目讓我深入理解了Canvas繪圖、遊戲循環和碰撞檢測等遊戲開發核心概念。後續更新計劃加入多人對戰模式和更多的視覺效果。

快打旋風遊戲工具

快打旋風戰鬥數據分析器

React Node.js Express MongoDB

這個工具為快打旋風遊戲玩家設計，能夠記錄和分析戰鬥數據，幫助玩家提升技術。主要功能包括招式幀數查詢、角色匹配分析、對戰錄像管理和技巧組合推薦。系統採用React構建前端界面，Node.js和Express作為後端API，MongoDB存儲用戶和遊戲數據。

開發過程中最大的挑戰是實現幀精確的招式判定系統，透過視頻處理技術和機器學習算法，成功實現了高達95%的準確率。這個工具已被當地電競社區採用，並獲得了積極反饋。

機器學習

台灣房價預測模型

Python TensorFlow Pandas Scikit-learn Matplotlib

基於十年台灣房價數據開發的預測模型，結合了線性回歸、隨機森林和深度學習網絡的集成模型。考量了超過20個影響房價的因素，包括地理位置、交通便利性、學區資源等。模型在測試集上達到了87%的準確率，可以為購房者和投資者提供較為精確的價格參考。

項目採用了Python生態系統的主要數據科學工具，從數據清洗、特徵工程到模型訓練和評估，完整展示了機器學習應用於實際問題的流程。代碼公開在GitHub上，並提供了詳細的技術報告和互動式視覺化界面。

Discord機器人

多功能學習助手機器人

Python Discord.py SQLite OpenAI API

為課堂和學習社群設計的Discord機器人，整合了多種學習輔助功能。主要特點包括自動組織學習小組、提醒作業截止日期、答疑系統、學習資源推薦和積分獎勵機制。機器人使用Discord.py框架開發，後端數據存儲使用SQLite，並整合了OpenAI API提供智能回答功能。

這個機器人已經在我們學校的5個不同Discord社群中使用，服務超過300名學生。從用戶反饋來看，作業提醒功能和智能問答系統特別受歡迎，幫助提高了學習效率和社群活躍度。
